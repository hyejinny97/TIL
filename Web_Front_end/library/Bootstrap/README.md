# 📚 Bootstrap

▪ Bootstrap이란?

반응형 웹을 빠르게 디자인하고 customize할 수 있게 해주는 front-end 오픈소스 toolkit이다. 

▪ Bootstrap의 특징

- Sass variables and mixins
- responsive grid system
- 다양한 prebuilt components
- 강력한 JavaScript plugins
