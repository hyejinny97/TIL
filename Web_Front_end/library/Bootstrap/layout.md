# ✔ Breakpoints
- viewport 크기에 따라 반응형 layout을 만들기 위해 너비를 customize한 것

> Breakpoints

- none: Extra small (<576px)

- `sm`: Small (>=576px)

- `md`: Medium (>=768px)

- `lg`: Large (>=992px)

- `xl`: Extra large (>=1200px)

- `xxl`: Extra extra large (>=1400px)
