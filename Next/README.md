# 📚 Next.js

▪ Next.js란?

Next.js는 서버 사이드 렌더링, 정적 웹 페이지 생성 등 리액트 기반 웹 애플리케이션 기능들을 가능케 하는 Node.js 위에서 빌드된 웹 풀스택 프레임워크이다.
