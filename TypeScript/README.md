# 📚 TypeScript

▪ TypeScript란?

TypeScript는 JavaScript의 모든 기능을 포함하면서 정적 타입을 지원하는 언어다.<br />
명시적인 정적 타입 지정은 개발자의 의도를 명확하게 코드로 기술할 수 있다.<br />
따라서, 코드의 가독성을 높이고 예측할 수 있게 하며 디버깅을 쉽게 한다.
