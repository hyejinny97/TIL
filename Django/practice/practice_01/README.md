- today-dinner 주소를 요청하면 랜덤으로 저녁 메뉴와 메뉴 이미지를 추천해주는 화면을 응답하는 프로그램을 개발합니다.
- 예시
  
  ![today-dinner 주소 완성본 이미지](img/today_dinner.png)

- lotto 주소를 요청하면 랜덤으로 로또 다섯 줄을 뽑아주는 화면을 응답하는 프로그램을 개발합니다.
- 예시

  ![lotto 주소 완성본 이미지](img/lotto.png)
