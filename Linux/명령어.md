# ✔ 명령어
> 디렉토리 관리에 관한 명령어
1. 현재 작업 중인 디렉토리 출력
    ```bash
    > pwd 
    ```
   - print working directory

2. 디렉토리 이동
    ```bash
    > cd {폴더명}
    ``` 
     - change directory
     - `.` : 현재 디렉토리, `..` : 상위 디렉토리

3. 목록
    ```bash
    > ls
    ```
    - list

4. 디렉토리 생성
   ```bash
   > mkdir {폴더명}
   ```
   - make directory
  
5. 파일 생성
    ```bash
    > touch {파일명}
    ```
   - `rm 파일명`: 파일 삭제하기
    - `rm -r 폴더명`: 폴더 삭제하기
