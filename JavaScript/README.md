# 📚 JavaScript

▪ JavaScript란?

JavaScript는 HTML 문서에 적용될 때, 웹사이트 상에서 동적 상호작용성을 제공할 수 있는 완전한 동적 프로그래밍 언어(dynamic programming language)이다.
